## Documentation Best Practices

### Do:
- Write notes same day
- Use objective language
- Include specific data
- Document clinical reasoning
- Sign and date all notes

### Don't:
- Use vague language ("did well")
- Leave sections blank
- Backdate documentation
- Copy/paste without individualizing

### Insurance Requirements:
- Date and time of service
- Location
- Service code
- Provider credentials
- Signature
- Activities linked to goals
- Measurable data
